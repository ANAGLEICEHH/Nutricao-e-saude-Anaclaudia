# Case Study: Identidade Visual e Presença Digital - Nutricionista Clínica

Este repositório documenta o processo de criação de branding e desenvolvimento web para a Dra. Ana Cláudia Carvalho. O objetivo principal foi transformar uma marca pessoal em uma autoridade digital, facilitando a conversão de pacientes através de um design estratégico e funcional.

## 🚀 Visão Geral do Projeto

O projeto consistiu em uma solução "End-to-End", onde atuei como Designer e Desenvolvedora Front-End para resolver a ausência de presença digital da cliente.

### 🎨 Branding & Identidade Visual
* **Criação de Logotipo:** Desenvolvimento de marca exclusiva com foco em "Nutrição e Saúde".
* **Paleta de Cores:** Seleção de tons de verde e vermelho (melancia), transmitindo frescor, saúde e profissionalismo.
* **Materiais Gráficos:** Elaboração de cartões de visita e banners de serviços para redes sociais, garantindo unidade visual em todos os pontos de contato.

### 💻 Desenvolvimento Web (Site Institucional)
* **Arquitetura:** Site responsivo (Mobile First) focado na jornada do paciente.
* **Conversão:** Implementação de botões de Call-to-Action (CTA) integrados diretamente ao WhatsApp para agendamento rápido.
* **UX/UI:** Interface limpa e intuitiva, priorizando a facilidade de leitura e navegação em dispositivos móveis.
* **SEO Local:** Estruturação de conteúdo focada em mecanismos de busca para visibilidade regional.

## 🛠️ Tecnologias e Ferramentas
* **Design:** Ferramentas de edição visual para criação de logos e artes gráficas.
* **Desenvolvimento:** HTML5, CSS3 e JavaScript.
* **Funcionalidades:** Integração de APIs de comunicação (WhatsApp).

* Estabelecimento de uma identidade de marca sólida e profissional.
* Redução do atrito no agendamento de consultas via integração direta com chat.
* Portfólio visual coerente, aplicado desde o cartão de visita físico até a plataforma digital.
